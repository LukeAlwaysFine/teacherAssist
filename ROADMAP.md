# teacherAssist — 路线图

## 已完成

### 核心功能
- [x] FastAPI 框架 + SQLite + JWT 认证 + 多 Provider LLM
- [x] 三引擎 STT（CPU / CUDA / API）+ 实时进度追踪
- [x] 课堂 Session 管理（CRUD + 知识点大纲 + 转录 + 分析 + 家长报告）
- [x] 前端 SPA（三步导航、历史记录、报告图片导出、全部中文化）
- [x] 报告图片生成（Pillow PNG）
- [x] 维护功能（清空上传音频）

### 知识点大纲 v4.0
- [x] 6 级结构：科目 → 教材 → 年级 → 册 → 单元/章 → 节 → 知识点
- [x] 168 单元、529 节、1642 知识点（6 套人教版教材）
- [x] 前端 6 级级联选择器（向后兼容旧格式）

### 部署
- [x] 原生 Windows 安装包（Inno Setup + 嵌入式 Python，用户双击即装）
- [x] Docker 容器化（Dockerfile + docker-compose，开发者备选）
- [x] 云端部署方案文档（Railway / 阿里云 / 腾讯云）
